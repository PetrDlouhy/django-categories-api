============
Installation
============

To use the Category model
=========================

1. Install django-categories::

   Follow installation instructions for https://github.com/jazzband/django-categories

1. Install django-categories-api::

    pip install django-categories-api

2. Add ``"categories"`` and ``"categories.editor"`` to your ``INSTALLED_APPS`` list in your project's ``settings.py`` file.
